# Decoding OM3
An open source project to fearlessly peer inside OM3. Contributions _extremely_ welcome! See rendered pages here (note case sensitive): [access-community-hub.github.io/decoding-om3/](https://access-community-hub.github.io/decoding-om3/)

# Get started...
All community members are welcome to contribute or correct errors/typos. To get write access, you need to create an issue and request access. You're also welcome to make a fork and put in a pull request.
